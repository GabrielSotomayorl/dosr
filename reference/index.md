# Package index

## Funciones principales

Estimaciones para una o más rondas de encuesta. Todas generan un reporte
en Excel con hoja consolidada y hojas de formato.

- [`obs_prop()`](https://gabrielsotomayorl.github.io/dosr/reference/obs_prop.md)
  : Calcula estimaciones de proporciones para diseños complejos
- [`obs_media()`](https://gabrielsotomayorl.github.io/dosr/reference/obs_media.md)
  : Calcula estimaciones de medias para diseños complejos
- [`obs_total()`](https://gabrielsotomayorl.github.io/dosr/reference/obs_total.md)
  : Calcula estimaciones de totales para diseños complejos
- [`obs_ratio()`](https://gabrielsotomayorl.github.io/dosr/reference/obs_ratio.md)
  : Calcula razones (ratios) para diseños complejos
- [`obs_cuantil()`](https://gabrielsotomayorl.github.io/dosr/reference/obs_cuantil.md)
  : Calcula estimaciones de cuantiles para diseños complejos

## Perfilado de variables binarias

Análisis de múltiples variables dicotómicas (0/1) en un único diseño de
encuesta.

- [`multi_bin()`](https://gabrielsotomayorl.github.io/dosr/reference/multi_bin.md)
  : Perfilar Múltiples Variables Dicotómicas con Criterios de Calidad

## Datos incluidos

Subconjuntos de la CASEN 2022 y 2024 con todas las observaciones y las
variables necesarias para reproducir los ejemplos del paquete.

- [`casen_2022`](https://gabrielsotomayorl.github.io/dosr/reference/casen_2022.md)
  : Encuesta CASEN 2022 — subconjunto de variables clave
- [`casen_2024`](https://gabrielsotomayorl.github.io/dosr/reference/casen_2024.md)
  : Encuesta CASEN 2024 — subconjunto de variables clave
